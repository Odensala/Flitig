# Flitig
Todo app with MVVM and the latest Android architecture components. Keep track of your daily todos with ease.
# Features
- Add, edit and delete todos
- Easily search for todos
# Key components
- MVVM
- Coroutines
- Flow
- Room
- Dagger Hilt
- Unit tests
# Planned features
- Material 3
